# Chriiisss03-HIT226-PreWork-week1

## Website Files

- `index.html` - main semantic HTML5 page
- `styles.css` - external CSS styling
- `script.js` - JavaScript interaction (theme toggle)
- `assets/validation.png` - validation screenshot file path

## How to Run

1. Open `index.html` in your web browser.
2. Click **Switch Theme** to test the JavaScript interaction.

## If `index.html` Does Not Load Correctly

If the page looks unstyled or the image/script does not load, you may be opening the file from a temporary ZIP path.

- Make sure all files are extracted and kept in the same project folder.
- Open this file directly from the project folder:
